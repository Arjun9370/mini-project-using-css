# mini-project-using-css

## Sidebar Menu
## Project Description

This is a mini front-end project created using HTML and CSS only.
It features a responsive sidebar navigation menu that opens and closes using a checkbox toggle technique (no JavaScript).

## Features

 Sliding sidebar menu

 Hamburger menu icon (open)

 Close icon (hide sidebar)

 Clean UI with Font Awesome icons

 Responsive layout

 Background image with full viewport height

 No JavaScript used (pure CSS)

## Technologies Used

HTML5

CSS3

Font Awesome Icons

Google Fonts (Poppins)
